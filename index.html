Тематический индекс Яндекса:

Технология Яндекса, которая определения авторитетность интернет-ресурсов с учётом качественной характеристики — ссылок на них с других сайтов.

PageRank - что это такое?

PageRank страницы — это алгоритм ссылочного ранжирования, показывает ее значимость для поисковиков. Определяется цифрой от 0 до 10.

Чем чаще ссылаются на источник ресурсы, тем более авторитетным он считается. Соответственно, PR будет выше.

Для чего его использует Google?

PageRank является фактором ранжирования сайтов в результатах поиска. Основная идея — чем больше кликов на страницу, тем она важнее и соответственно попадает в число первых результатов поиска.

Google учитывает персональные данные пользователей для выдачи только тех результатов, которые ему нужны. Это явление имеет негативные стороны, например, если пользователь критикует президента в социальных сетях, то на запрос о гражданском мнении о президенте он получит только результаты с негативными характеристиками последнего, вообще на любой запрос пользователь будет получать не противоречащую его мнению информацию, что разумеется не очень хорошо влияет на его гражданское мнение.

Сейчас в Google возможна функция отключения просмотра персональных результатов при просмотре сайтов.

Файл robots.txt и для чего нужен:

Файл robots.txt используется управления трафиком поисковых роботов на вашем сайте. C его помощью также можно исключить контент из результатов поиска Google.

Правила записи

Все блоки правил начинаются с директивы User-agent, в которой указывается название робота, для которого задается правило. Запись вида User-agent: * означает, что правило задается для всех поисковых роботов.

Существует большое множество команд которые позволяют управлять роботом. Например, можно разрешить или наоборот запретить к отдельным разделам, задать частоту работы.

Назначениие карты сайта и ее создание:

Карта сайта представляет собой список страниц сайта для поисковых систем или пользователей и аналогична разделу Содержание обычной книги. Используется как элемент навигации, показывает взаимосвязь между страницами сайта. Это полный перечень разделов всех страниц в иерархическом порядке.

Создать карту сайта можно двумя способами. Первый – сделать sitemap.xml можно с помощью внешнего онлайн-генератора. Второй – понадобится модуль или внутренний генератор для конкретной CMS. Рассмотрим оба способа.

Файл sitemaps:

Sitemaps представляет собой XML-файлы с информацией для поисковых систем страницах веб-сайта, которые подлежат индексации. Sitemaps могут помочь поисковикам определить местонахождение страниц сайта, время их последнего обновления, частоту обновления и важность относительно других страниц сайта для того, чтобы поисковая машина смогла более разумно индексировать сайт.
